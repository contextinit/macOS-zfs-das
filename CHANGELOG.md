# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-XX

### Added
- Initial release of macOS ZFS DAS solution
- Auto-mount script with encryption key management
- Monthly maintenance script with automated scrubbing
- SwiftBar monitoring plugins (basic and advanced)
- Time Machine setup helper
- Complete documentation suite
- LaunchDaemon configurations
- Example configurations for common use cases

### Features
- AES-256 encryption support
- RAID-Z1 (RAID-5) configuration
- Automatic pool import on boot
- Real-time health monitoring via SwiftBar
- Time Machine integration for multiple Macs
- Monthly automated maintenance and scrubbing
- Comprehensive troubleshooting guides
